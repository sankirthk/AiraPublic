## Capture Guidance

- Prefer clean PNG screenshots at 2x resolution
- Use short crops instead of giant desktop captures
- Keep the desktop background neutral so the overlay reads clearly
- Show one idea per image
- If you record video, keep it under 30 to 45 seconds for the README demo
- Show the app in real use, not only settings panels

## Suggested Order

1. Hero image

3. Overlay modes
4. Session controls
5. Accessibility controls
6. Menu bar access
7. Demo video
